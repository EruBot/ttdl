# TikTok Downloader Bot (No Watermark)

Repository ini berisi bot Telegram untuk mengunduh video TikTok tanpa watermark.

## Fitur
- Download video TikTok tanpa watermark
- Cepat dan ringan
- Support berbagai link TikTok

## Tujuan
Memberikan solusi sederhana untuk mengunduh konten TikTok tanpa watermark langsung dari Telegram.
